# ReconStrike

**ReconStrike** is a Linux CLI-based offensive recon tool that combines powerful network and Wi-Fi scanning capabilities using `nmap`, `aircrack-ng`, and related utilities. Inspired by tools like Nmap and Wifite, ReconStrike aims to unify LAN and Wi-Fi reconnaissance into one streamlined interface.

---

## 🚀 Features

- ✅ **LAN Scanner**: Fast port scanning using Nmap (`-sS -T4`)
- ✅ **Wi-Fi Scanner**: Wireless interface detection, monitor mode activation, and scanning via `airmon-ng` and `airodump-ng`
- 🧠 Simple and interactive terminal interface
- 🐧 Optimized for Linux systems (Kali, Ubuntu, Parrot, etc.)
